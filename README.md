## Desafio para o pessoal de pelotas

### Descrição
  ### Frontend
  O desafio consiste na elaboração e implementação de um projeto em react.js que seja responsável por implementar o frontend de uma aplicação
  O projeto consistem em 3 telas:
  * Login (email,senha)
  * Cadastro (email,senha,nome,telefone)
  * Tela principal:
    * Exibição de um mapa
    * Cadastro de reservatório (nome,bounding box do poligono)
    * Exibir Reservatorios no mapa
    * Botão sair
      
  ### Backend
  Desenvolva uma API REST API que implemente os casos de uso da aplicação frontend
  * Endpoints:
    * Login
    * Cadastro
    * Cadastro de reservatório
    * Listar reservatório
### Dúvidas
  Chama no zap!
